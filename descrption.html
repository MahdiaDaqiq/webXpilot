AN EVOLVING XPILOT FUZZY AGENT


The project consists of creating an xpilot fuzzy agent and make the fuzzy sets adaptable to various situations, maps, opponents. The fuzzy agent utilizes a genetic algorithm to evolve over time in the goal of staying alive as long as possible. Staying alive involve the need for killing, hiding, avoiding walls and avoiding bullets. The ultimate goal of the agent is to win, by being the last one staying no matter what it is facing because of its ability to adapt and its use of fuzzy logic.




HYPOTHESIS

The hypothesis of the project was that the evolving agent that utilizes a fuzzy logic would be able to adjust its fuzzy sets depending on its opponents to eventually be able to learn a winning strategy. A winning strategy is just a strategy that kills the other agent or agents that it is facing in battle by either killing them or staying alive longer than its opponent, who might end up crashing into a wall. The belief was that if the agent plays a very aggressive agent or a relatively defensive agent, it would be able to adapt itself by creating totally different fuzzy sets.

FUZZY LOGIC

“Fuzzy logic is determined as a set of mathematical principles for knowledge representation based on degrees of membership rather than on crisp membership of classical binary logic A fuzzy set is a class of objects with a continuum of grades of membership. Such a set is characterized by a membership (characteristic) function which assigns to each object a grade of membership ranging between zero and one. The notions of inclusion, union, intersection, complement, relation, convexity, etc., are extended to such sets, and various properties of these notions in the context of fuzzy sets are established.” (Zadeh, 1965) To learn more about FL _link_ Fuzzy-Logic




GENETIC ALGORITHMS

Genetic algorithms is a heuristic search that is inspired by Charles Darwin’s theory of natural selection. This algorithm exhibits the methods of natural selection where the individuals with the heights fitness are chosen for reproduction in order to produce offspring of the next generation.
Genetics algorithms reflect Natural Selection through five phases: Initial Population, fitness function, Selection, Crossover and Mutation
Initial Population is a set of chromosomes (individuals) where each chromosome is a solution for the problem you to be solved.
Fitness function is a function where each individual can be calculated based on their desirable traits. The individuals who carry desirable traits will get a higher fitness score than the individuals who do not.
Selection: is a process where two chromosomes are selected to reproduce. The individuals selected are random but the higher the chromosome’s fitness score will allow them to have a higher chance of getting selected during the selection process.
Crossover: is when the two selected chromosome combines through different methods to form the child chromosome.
